D.I.S.C.O Preliminary Program V7.6 ROUND 2 EDITOR

추가 기능
- 자동으로 생성된 2차 진출자 명단을 관리자가 수정 가능
- 진출자 삭제
- 1차 전체 참가자 중 탈락자를 다시 진출자로 추가
- 명단 순서를 위/아래 버튼으로 변경
- 삭제·추가·순서 변경 시 2차 ORDER와 조 자동 재배정
- 예: A-9 삭제 → 기존 A-10이 A-9 → 기존 B-1이 A-10
- 관리자가 '명단 최종 확정 · 2차 채점 열기'를 눌러야만 져지 채점 화면 활성화
- 최종 확정 전에는 is_ready=false 상태로 유지
- 최종 선택 명단도 1차 ARCHIVE에 함께 기록

추가 SQL 실행은 필요 없음.
V7.5 SQL을 이미 실행했다면 그대로 사용 가능.
